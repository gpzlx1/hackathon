What is Convolutional Neural Networks?



A convolution neural network (CNN) is a type of artifical neural network used in image recognition area and so on. As we know, it is very simple for human or animinals to detect and identify objects. But what should the computer do to identify a number 8? Even if 8 is in the case of a two-digit number array, we still can abstract it out. To solve this proble, smart scientists have proposed and implemented an effective way for computer implementations to perform pattern recognition, which is known convolution neural network.





A complete Convolutional Neural Network

This picture shows how CNN recognizes a car. The structure of CNN may look very complicated, but in reality only four operators are combined to form this CNN. We will introduce these four components in order.




Convolution Layer



The first operation is a convolution operation. In short, its function is to scan pictures (in fact, it is Matrix element-multiplication) and extract features. Like this gif, the convolution kernel constantly scans some or all of the images, constantly proposing features. The convolution layer has two parameters, one is size, which determines the size of the scan area; the other is the stride, which determines the distance between adjacent scan areas.




Pooling Layer





The role of the pooling layer is to magnify the features. Its behavior pattern is consistent with the convolutional layer.
There are also two parameters, size and stride. However, it does not multiply but takes the maximum operation.
In this way, CNN can respond more strongly to some certain features.




Activation Layer



The activation layer determines whether the network responds to features. CNN is able to extract a lot of features, but only a few features can make a big difference to our recognition. Just like looking for a zebra at the zoo, the main feature is the coat color. The function of acitvation layer is filtering out useful features. There is no paramter but the method to activate.

Different Activation Function






Fully-Connected Layer



The fully connected layer can aggregate features and aggregate all the features together to identify what the object is. Just like humans, CNN extracts all features through the fully connected layer and identifies the target based on the combination of features. Its parameter is the number of features extracted
